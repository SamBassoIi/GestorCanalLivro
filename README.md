# 📚 Sistema de Gestão de Venda de Livros

Aplicação web interativa desenvolvida para o controle de inventário e gestão de vendas de um acervo literário. Projeto focado na manipulação avançada de DOM e persistência de dados local.

## 🚀 Tecnologias Utilizadas
* **HTML5 & CSS3** - Estruturação e design da interface.
* **JavaScript (ES6+)** - Lógica de negócios e interatividade.
* **LocalStorage** - Persistência de dados no navegador do usuário sem necessidade de banco de dados externo.

## 🌟 Funcionalidades
* Cadastro detalhado de obras literárias (título, autor, gênero, quantidade).
* Filtro de buscas em tempo real.
* Controle de estoque dinâmico (entrada e saída de livros).
* Interface limpa e intuitiva para o usuário final.
